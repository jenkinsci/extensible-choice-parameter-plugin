Groovyスクリプトを記述します。スクリプトが返した配列を選択肢として利用できます。

  • String型でない要素は toString() で文字列に変換します。
  • null要素は無視します(選択肢に表示しません)
  • 例外が発生したり、配列でない値を返した場合、選択肢は空になります。

スクリプトの例:

Example1----------------------------------------------------------------------------- // 固定の選択肢を返す return ["choice1", "choice2", "choice3"]; Example2----------------------------------------------------------------------------- // 1,2,3,...,10　を選択肢として表示 return 1..10; Example3----------------------------------------------------------------------------- // 過去一週間の日付 return (6..0).collect{new Date().plus(-it).format("yyyy-MM-dd")};
Groovy スクリプトのセキュリティについては Script Security Plugin を参照してください。
セキュリティの都合上、設定画面でのスクリプト実行には以下の制限があります。
  • Sandbox を使用するスクリプトのみが実行できます。
  • 追加のクラスパスは参照されません。
上記の制限のため、設定画面で以下の動作に制限がある場合があります。
  • デフォルト値の選択肢
  • スクリプトのテスト実行
